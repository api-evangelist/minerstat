# Minerstat (minerstat)
Minerstat mining pools API is a public API that allows you to obtain basic information about different mining pools that are listed on minerstat. Pool data updates approximately once per minute and is throttled to roughly 12 requests per minute. Authentication uses an API key sent via the X-API-Key header or key query parameter.
